# Strava Setup Guide

Follow these steps to connect your Strava account to Movara.

## 1. Log in to Strava

Go to **www.strava.com** and log in to your account (or create one if you don't have one yet).

## 2. Open your API application settings

- Click your **profile icon** (top right) and select **Settings**
- In the left sidebar, click **My API Application**
- Or go directly to **www.strava.com/settings/api**

## 3. Create an API application (if you don't have one)

If you don't have an API application yet, create one. The other fields can be anything, but these two are important:

- **Website** must be set to `movara.fit`
- **Authorization Callback Domain** must be set to `movara.fit`

## 4. Copy your credentials

On the API application page, copy the **Client ID** and **Client Secret**.

## 5. Enter credentials in Movara

Open Movara, go to the **Profile** tab, and find the **Strava** section. Paste your Client ID and Client Secret into the input fields and tap **Save**.

## 6. Connect with Strava

Tap **Connect with Strava** and authorize the app. Movara will import your last 90 days of activities automatically.
